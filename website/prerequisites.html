Workshop Prerequisites

To ensure a successful workshop experience, please review and complete the prerequisites for your chosen workshop level. Each level builds upon the previous one, so higher-level workshops include all prerequisites from lower levels.

Basic Workshop Prerequisites

The Basic Workshop is designed to be accessible to most developers with minimal setup. Here's what you'll need:

Required Accounts

  • GitHub Account: Free account with access to GitHub Copilot (trial or subscription)
  • Visual Studio Code: Latest stable version

Development Environment

  • Operating System: Windows 10/11, macOS 10.15+, or Linux (Ubuntu 20.04+ recommended)
  • Hardware: 8GB RAM minimum, 16GB recommended
  • VS Code Extensions: GitHub Copilot, GitHub Copilot Chat
  • Internet Connection: Stable broadband connection

Knowledge Prerequisites

  • Programming: Basic knowledge of at least one programming language
  • Git: Basic understanding of Git commands (clone, pull, push)
  • VS Code: Basic familiarity with the editor

Setup Instructions

  1. Install Visual Studio Code from https://code.visualstudio.com/
  2. Sign in to GitHub within VS Code
  3. Install the GitHub Copilot extension from the VS Code marketplace
  4. Install the GitHub Copilot Chat extension from the VS Code marketplace
  5. Clone the workshop repository: git clone https://github.com/paulasilvatech/Code-AI-Dev.git

Intermediate Workshop Prerequisites

The Intermediate Workshop builds on the Basic Workshop and requires additional setup:

Required Accounts

  • All Basic Workshop accounts
  • Azure Account: Free trial or existing subscription

Development Environment

  • All Basic Workshop requirements
  • Azure CLI: Latest version
  • Node.js: Version 16+ (LTS recommended)
  • Python: Version 3.8+
  • Docker: Latest stable version

Knowledge Prerequisites

  • All Basic Workshop knowledge
  • Cloud Concepts: Basic understanding of cloud services
  • CI/CD: Familiarity with continuous integration/deployment concepts
  • Security: Basic understanding of application security principles

Setup Instructions

  1. Complete all Basic Workshop setup steps
  2. Install Azure CLI from https://docs.microsoft.com/en-us/cli/azure/install-azure-cli
  3. Install Node.js from https://nodejs.org/
  4. Install Python from https://www.python.org/downloads/
  5. Install Docker from https://www.docker.com/products/docker-desktop
  6. Log in to Azure CLI: az login
  7. Install workshop dependencies: cd Code-AI-Dev && npm install

Advanced Workshop Prerequisites

The Advanced Workshop is designed for experienced developers and requires a more comprehensive setup:

Required Accounts

  • All Intermediate Workshop accounts
  • Azure DevOps: Organization and project
  • GitHub Enterprise: If using in enterprise context

Development Environment

  • All Intermediate Workshop requirements
  • Hardware: 16GB RAM minimum, 32GB recommended
  • Kubernetes CLI (kubectl): Latest version
  • Terraform: Latest version
  • Azure DevOps CLI: Latest version
  • GitHub CLI: Latest version

Knowledge Prerequisites

  • All Intermediate Workshop knowledge
  • DevOps: Strong understanding of DevOps principles
  • Cloud Architecture: Familiarity with cloud architecture patterns
  • Containers: Experience with containerization
  • Infrastructure as Code: Basic understanding of IaC concepts

Setup Instructions

  1. Complete all Intermediate Workshop setup steps
  2. Install Kubernetes CLI: https://kubernetes.io/docs/tasks/tools/
  3. Install Terraform: https://www.terraform.io/downloads.html
  4. Install Azure DevOps CLI: az extension add --name azure-devops
  5. Install GitHub CLI: https://cli.github.com/
  6. Create Azure resources using the provided script: ./scripts/setup-advanced-workshop.sh

Enterprise Track Prerequisites

The Enterprise Track is designed for teams and organizations implementing AI-assisted development at scale:

Required Accounts

  • All Advanced Workshop accounts
  • GitHub Enterprise: Organization-wide access
  • Azure Enterprise Agreement: Or equivalent enterprise cloud subscription

Team Requirements

  • Development Team: Multiple developers (3+ recommended)
  • Platform Team: DevOps/platform engineers
  • Security Team: Security engineers or representatives
  • Management: Engineering managers or technical leaders

Environment Requirements

  • All Advanced Workshop requirements for each participant
  • Shared Environment: Azure subscription with contributor access
  • Collaboration Tools: Microsoft Teams, Slack, or equivalent
  • Meeting Space: Physical or virtual room for team discussions

Knowledge Prerequisites

  • All Advanced Workshop knowledge
  • Enterprise Architecture: Understanding of enterprise systems
  • Governance: Familiarity with IT governance principles
  • Change Management: Experience with organizational change
  • ROI Analysis: Ability to measure and evaluate technology investments

Preparation Tasks

  1. Complete all Advanced Workshop setup steps for each participant
  2. Conduct pre-workshop assessment (provided separately)
  3. Gather current development metrics for baseline comparison
  4. Review organizational policies related to AI tools
  5. Identify pilot projects for AI implementation
  6. Schedule follow-up sessions for implementation planning

Verification Checklist

Before attending the workshop, verify that you can complete these tasks:

  • Open VS Code and confirm GitHub Copilot is active (icon in status bar)
  • Create a new file and verify Copilot suggestions appear
  • Clone the workshop repository and open it in VS Code
  • Run the verification script: ./scripts/verify-setup.sh
  • For Intermediate+ workshops: Confirm Azure CLI login works
  • For Advanced+ workshops: Verify Kubernetes and Terraform installations

If you encounter any issues, please refer to the FAQ or contact the workshop organizers for assistance.
